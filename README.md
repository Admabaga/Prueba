Servicio Rest: `POST http://localhost:8080/empleados`

```JSON
{

    "apellidos": "Barrera Garcia",
    "numeroDocumento":"1510810",
    "fechaNacimiento":"1995-10-14",
    "fechaVinculamiento":"2018-11-25",

    "salario": 20000

```

Formato Fecha: `yyyy-mm-dd`
